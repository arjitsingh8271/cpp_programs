/*
Access Modifiers:
The access modifiers of C++ are allows us to determine which class 
members are accessible to other classess or functions, and which not

1. Public:
	The public member are accessible from any part of the program.

2. Private:
	The Private members can only be accessed from within the class.
	However, friend class & friend function can access private members.

3. Protected:
	The Protected members can be accessed within the class and from
	the derived class or child class.


By default the access modifier is Private

*/
