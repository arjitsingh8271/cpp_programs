# How to Run Programs

## CMD in Windows
\>cd Desktop\
\Desktop> cd "c++ programming"\
\Desktop/c++ programming> **g++ fileName.cpp**
\Desktop/c++ programming> **.\a.out**

## Terminal in Linux/Mac
/$ cd Desktop/
/Desktop$ cd "c++ programming"/
/Desktop/c++ programming$ **g++ fileName.cpp**
/Desktop/c++ programming$ **./a.out**
