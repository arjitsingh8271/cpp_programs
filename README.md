# How to Run Programs

## CMD in Windows
\>cd Desktop\<br>
\Desktop> cd "c++ programming"\<br>
\Desktop/c++ programming> **g++ fileName.cpp**<br>
\Desktop/c++ programming> **.\a.out**<br>

## Terminal in Linux/Mac
/$ cd Desktop\<br>
/Desktop$ cd "c++ programming"\<br>
/Desktop/c++ programming$ **g++ fileName.cpp**<br>
/Desktop/c++ programming$ **./a.out**<br>
